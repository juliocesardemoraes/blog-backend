## What is it

NESTJS project for a rest api backend server.

## How to run it

```bash
# install packages
npm install

# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
